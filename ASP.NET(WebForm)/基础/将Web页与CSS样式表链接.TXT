将 Web 页与 CSS 样式表链接请参见
使用 CSS 样式 | 级联式样式表 (CSS) 介绍 | 生成 CSS 样式 | 创建外部 CSS 样式表 | 将 CSS 样式应用到另一个 Web 页
当 CSS 样式在外部 CSS 样式表中时，对其所做的任何更改都会在所有链接到样式表的 Web 文档中反映出来。可将许多文档链接到同一 CSS 外部样式表。这样就不必在多个文档中重复创建 CSS 样式，从而消除了许多冗余的工作，并能确保所有 Web 页上的样式保持一致。

在设计视图中添加到 CSS 样式表的 LINK 

在设计视图中，打开要链接到样式表的 HTML 文档进行编辑。 
从“格式”菜单中选择“文档样式”。 
注意   当在“设计”视图中打开 HTML 文档并将其 targetSchema 属性设置为支持 HTML 4.0 或更高版本的 Web 浏览器时，“格式”菜单上的“文档样式”选项变得可用。
从“格式”菜单中选择“文档样式”。 
在 HTML 视图中添加到 CSS 样式表的 LINK 

在 HTML 视图中打开 HTML 文档。 
从解决方案资源管理器中将所需样式表拖到活动 HTML 文档的 <HEAD> 节。这样就插入了一个新的 LINK 元素： 
<LINK REL="stylesheet" TYPE="text/css" HREF="MyStyles.css">
MyStyles.css 是外部 CSS 样式表的名称。 

根据需要，编辑 HREF 属性中指定的样式表文件的路径。 
从“文档样式”窗口添加到 CSS 样式表的 LINK 

在设计视图中打开 HTML 文档。 
从“格式”菜单中选择“文档样式”。 
出现“文档样式”窗口。 

从“文档样式”窗口的顶部选择“添加样式链接”。 
出现“选择样式表”对话框。 

转到 CSS 样式表文档，选择“打开”。 
链接的路径出现在“文档样式”窗口中。 

根据需要（例如，指定样式表的相对路径），切换到“HTML”视图并在新的 LINK 元素中编辑样式表文件的路径。
